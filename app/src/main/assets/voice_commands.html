Voice Commands

If voice input is enabled, you can use the following voice commands.

Main board screen

• Solution word / Put word - Enter word into the grid. E.g. "put potato".
• Letter l - Enter a single letter l into the grid. E.g. "letter b".
• Delete - Delete at the cursor.
• Clear - Clear the current selected word.
• Number n - Jump to clue number n. E.g. "number 27".
• Toggle - Change selected clue direction.
• Next/previous/jump - Go to the next/previous/random clue.
• Left/right/up/down - Same as arrow keys.
• Clues - Go to the clue list screen.
• Notes - Go to the current clue's notes page.
• Back - Same as back button.
• Announce - Announce the current clue and board contents. Will use accessibility service if running, else the available text-to-speech service.
• Help - Ask ChatGPT for help with current clue. Requires a network connection and Chat GPT to be configured in interaction settings.

Notes screen

• Solution word / Put word - Enter word into the grid. E.g. "put potato".
• Letter l - Enter a single letter l into the grid. E.g. "letter b".
• Delete - Delete at the cursor.
• Clear - Clear the current selected word.
• Scratch word - Enter word into scratch. E.g. "scratch potato".
• Scratch clear - Clear the scratch box.
