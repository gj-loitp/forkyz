Puzzle List Help

Browse the list of puzzles and select one to play by touching it. You can also "Long Touch" a puzzle to delete it, or move it to the "Archive" view. You can also press the floating plus button to import puzzles.

Floating + Button

Pressing the floating plus button gives the following options:

• Online Sources - Opens a page with links to websites where puzzles can be downloaded
• Import Puzzle -Import a puzzle from your device. Forkyz will try to recognise the file format automatically. Puzzles can be downloaded from the online sources, or using external tools such as xword-dl.
• Download Puzzles - Use in-app direct download from sources configured in settings.

Menu Options

Pressing the Menu button on this screen provides the following options:

• Help - Displays this screen
• Settings - Displays the Application settings. Here you can select puzzles to download and customize the behavior of the Cleanup operation.
• Archive/Crosswords - Toggles between the regular list of crosswords and the archive folder
• Cleanup - Moves completed and older puzzles to the Archive folder. If you wish to delete, instead of moving them to the archive folder, you can specify this in the settings. The puzzle files are not that large, though, and won't take up a significant amount of space on your external storage device.
